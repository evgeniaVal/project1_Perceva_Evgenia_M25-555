# Лабиринт сокровищ

Текстовая консольная adventure-игра на Python.  
Игрок исследует лабиринт, перемещается между комнатами, собирает предметы,
решает загадки и пытается добраться до сокровищ.

Проект выполнен в рамках учебного задания и реализован без использования классов -
только функции, словари и базовые конструкции Python.

## Установка

Проект использует Poetry для управления зависимостями.

1. Клонируйте репозиторий:
```bash
git clone git@github.com:evgeniaVal/project1_Perceva_Evgenia_M25-555.git
cd project1_Perceva_Evgenia_M25-555
```
2. Установите зависимости:
```bash
make install
```
## Запуск игры
Запуск осуществляется через:
```bash
make project
```
После запуска игра выведет приветственное сообщение и описание стартовой комнаты.
## Управление
Доступные команды в игре:
* ```go <direction>``` - перейти в направлении (```north```, ```south```, ```east```, ```west```)
* ```north | south | east | west``` - короткие команды перемещения
* ```look``` - осмотреть текущую комнату
* ```take <item>``` - подобрать предмет
* ```use <item>``` - использовать предмет из инвентаря
* ```inventory``` - показать инвентарь
* ```solve``` - попытаться решить загадку
* ```help``` - показать список команд
* ```quit``` - выйти из игры
